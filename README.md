# CineLink
Django와 React로 구성된 영화 예매 사이트

--

## Page
- 홈페이지(/home)
- 로그인(/login)
- 마이페이지(/mypage)
- 패스워드 찾기(/password)
- 아이디 찾기(/findID)
- 회원가입(/signup)
- 영화 취향 고르기(choosingMovie) --> 팝업으로 할지 미 결정
